# Sistema de Biblioteca - PI

**Status do Projeto:** Em desenvolvimento.

**Tecnologias Aplicadas:**
* Java (Interface Swing)
* JDBC para persistência de dados
* MySQL/MariaDB (via XAMPP)

**Time de Desenvolvedores:**
* [Seu Nome Aqui]

**Objetivo do Software:**
O sistema serve para digitalizar o controle de acervo de uma biblioteca, permitindo o cadastro organizado de livros e leitores em um banco de dados real.

**Funcionalidades do Sistema:**
* Autenticação de usuário (Login).
* Cadastro definitivo de livros no banco de dados.
* Cadastro definitivo de leitores no banco de dados.
* Interface gráfica intuitiva com navegação por Dashboard.
